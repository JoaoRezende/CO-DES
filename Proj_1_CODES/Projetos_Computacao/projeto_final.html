Projeto Final de Design de Software

O projeto final da matéria Design de Software consistiu em desenvolver um projeto que envolvesse programação com o Python . Era possível realizar desde jogos, aplicativos até “webapps”. Existiram projetos variados que usaram, como por exemplo, tkinter (interface gráfica), firebase (armazenagem na nuvem) e html. Nesse projeto a dificuldade foi conseguir fazer o que se desejava, ja que a maioria das etapas era necessário um conhecimento de programação que não foi aprendido na sala de aula. Somente com pesquisas, consultas com site e com o professor era possível alcançar o objetivo. Alem disso foi usado o trello para dividir as tarefas igualmente.

Nesse projeto foram produzidos diferentes softwares, de acordo com cada grupo, porem não houve uma fabricação de um produto assim como não envolveu eltronica. A maioria dos projetos teve o design voltado para o usuário. O projeto nao envolveu escrita cientifica. Foi realizado no final do 1 semestre, com aproximamente 5 semanas*** de duração.
